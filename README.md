# MIN Growth Marketing Dashboard

Live dashboard: **https://averyl-mhk.github.io/min-growth-dashboard/**

Interactive HTML dashboard for Meyer India (MIN) marketing performance.
Tracks spend, attributed revenue, ROAS, and POS across Amazon, Meta, and Flipkart.

---

## Monthly update workflow

1. Drop Akash's raw export files into `raw_exports/YYYY_MM/`
2. Run the processor:
   ```
   python processor.py --month 2026_03
   ```
3. Verify `data.json` looks right, then push:
   ```
   git add data.json
   git commit -m "Add Mar 2026 data"
   git push
   ```
4. Dashboard at the live URL updates within ~60 seconds.

---

## Repo structure

```
index.html       Dashboard (do not edit manually)
data.json        Current data — auto-generated by processor.py
processor.py     Reads raw exports, writes data.json
raw_exports/     Drop monthly export files here (gitignored — stays local)
```

## Files needed each month (from Akash)

| File | Source |
|------|--------|
| Amazon campaign report | Amazon AMS → Reports → Campaign |
| SP Search Term report | Amazon AMS → Reports → Search Term |
| SB Search Term report | Amazon AMS → Reports → Search Term |
| SD Targeting report | Amazon AMS → Reports → Targeting |
| Meta campaign export | Meta Ads Manager → Export |
| Flipkart campaign export | Flipkart Seller Hub → Ads → Reports |
| POS totals (manual) | Amazon / Shopify / Flipkart sales dashboards |
